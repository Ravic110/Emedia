Introduction au HTML

Le HTML (HyperText Markup Language) est le langage de base du Web. Il sert à structurer le contenu d'une page : titres, paragraphes, liens, images, listes, formulaires, etc.

Structure minimale

Voici une page HTML complète avec les éléments de base :

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Ma première page</title>
  </head>
  <body>
    <h1>Bonjour le monde !</h1>
    <p>Ceci est une page HTML de démonstration.</p>
  </body>
</html>

Exercice : éditeur HTML

Modifiez le code ci-dessous, puis cliquez sur Exécuter pour voir le résultat. Vous pouvez changer les titres, ajouter des listes, ou insérer une image.

Prochaines étapes

Maintenant que vous savez créer une page HTML, passez à la leçon CSS pour apprendre à la rendre jolie et responsive.
