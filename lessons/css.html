Introduction au CSS

Le CSS (Cascading Style Sheets) permet d'ajouter du style aux pages HTML. Il transforme une structure en quelque chose de plus lisible et attractif.

Comment lier du CSS à une page HTML

Vous pouvez lier un fichier CSS à votre page en ajoutant une balise <link> dans la section <head>. 

<link rel="stylesheet" href="styles.css" />
