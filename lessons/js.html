Introduction à JavaScript

JavaScript permet de rendre les pages web interactives. Il peut modifier le contenu d'une page, réagir aux clics, et communiquer avec des serveurs.

Exemple : afficher un message

En JavaScript, on peut afficher du texte dans la console du navigateur ou modifier le contenu d'un élément HTML. 

const message = 'Bonjour depuis JavaScript!';
console.log(message);
document.getElementById('mon-titre').textContent = message;
